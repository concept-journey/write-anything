# Write Anything

Write Anything is a 글 저장소 for turning 아무 글감 into 공개 가능한 글.

It is built around a 글 정리 대화상대: a place to leave 덜 정리돼도 된다 thoughts, 캐묻기 them one by one, 함께 모양잡기 the right shape, and keep the result as a 글 묶음.

## Places

- `seeds/`: 글감 노트. Leave words, fragments, questions, observations, feelings, and experiences here before they become a 글 묶음.
- `pieces/`: 마크다운 글. Keep 글 묶음 here with a 열어보고 싶은 제목, 핵심 문장, 정리된 글, and 건네는 말.

## Flow

글감 남기기 -> 캐묻기 -> 함께 모양잡기 -> 글로 정리하기 -> 다듬기 -> 건네는 말 만들기

The work does not fail when it stops early. 멈춰도 남는다.

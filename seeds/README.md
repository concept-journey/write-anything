# Seeds

`seeds/` holds 글감 노트.

Use this place for words, fragments, questions, observations, feelings, and experiences that are not yet 정리된 글. They can be 덜 정리돼도 된다, and they can later split into several 글 묶음 or merge with other 글감.

Name files with 날짜-짧은-이름, such as `2026-05-31-writing-is-thinking.md`.
